# Geocoding Service

Geocoding Service es una servicio diseñado para identificar la posición geografica de una dirección y establecer a cuál ruta debería estar asignada.

## Funcionalidades

* Obtener la posición geografica de una dirección, en cualquier formado. Ejm: Calle, Cl, Call, Cll, Carrera, Cra, Kra, Cr, etc...
* Establecer la cobertura a la cuál debería ir asignado un paquete, apartir de una lista de coberturas permitidas y la dirección del cliente.
